# Product Overview

Swipe Check is a daily personality tracker that makes MBTI personality assessment fun and effortless. Users answer 3 quick questions per day through a swipe-based interface (like dating apps) to track their personality type over time.

## Core Concept

- **Daily engagement**: 3 questions/day (10 on first day for onboarding)
- **Swipe interface**: Quick, casual, game-like experience - swipe right to agree, left to disagree
- **Personality tracking**: Based on 16 personalities (MBTI) framework with 4 dimensions (E/I, S/N, T/F, J/P)
- **Rolling window**: Personality type calculated from last 7 days of responses (all days weighted equally)
- **Streak system**: Track consecutive days with milestone celebrations

## Key Features

- Current personality type display with 4-letter code and percentage breakdowns
- Streak tracking (current and longest)
- PWA support for mobile-first experience
- Authentication via Better Auth
- Dashboard showing type, streak, and participation stats

## Design Philosophy

- Mobile-first PWA with web support
- Playful, game-like aesthetic (not a clinical test)
- Smooth swipe gestures optimized for mobile
- Frictionless quiz experience
- Delightful animations for celebrations and type changes
- Vibrant colors, rounded corners, fun micro-interactions
